PROJECT TITLE: Manajemen Penumpang Bus 
PURPOSE OF PROJECT: Tugas 1 pratikum PBO 2020
VERSION or DATE: 11 November 2020
HOW TO START THIS PROJECT: 1. Clone Repository
			                     2. Buka folder hasil download tersebut menggunakan IDE(disarankan memekai blue j)
			                     3. Jika menggunakan blue j, cukup dengan menggunkan file package.bluej 

How to running code : 	   1. Pada kelas Main klik kanan, kemudian pilih method yang diinginkan.
			                     2. Method menu() untuk mensimulasikan pogram  berdasarkan tugas 1. 
			                     3. method main() untuk mensimulaikan pogram berdasarkan objek yang diinginkan
                           4. jika menggunakan IDE lainnya seperti VC Studio method menu(), bisa diubah ke bentuk main(String []arrgs) untuk mensimulasikan pogram 
                              berdasarkan tugas 1 dan methdo main sebelumnya bisa dihapus.

DESCRIPTION : 
1. Bus hanya memiliki 6 kursi penumpang, yang terdiri dari 4 kursi penumpang    biasa dan 2 kursi prioritas.
2. Kursi prioritas hanya dapat diisi oleh orang-orang berikut :
   Lansia : umur lebih dari 60 tahun
   Anak-anak : umur kurang dari 10 tahun
   Ibu hamil
3. Penumpang biasa dilarang duduk di kursi prioritas
4. Penumpang prioritas boleh duduk di kursi biasa.
5. Penumpang tidak bisa lagi menaiki bus jika bus sudah penuh
